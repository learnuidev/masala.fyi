# Curry

Next Gen Food + Weight tracker app

## Tech Stack

- nextjs - Framework
- vercel - Hosting
- dynamodb - Storage
- cognito - Auth
- resend - Email
- myelin.dev - AI Translations
- polar_sh - Payments
- step functions - Background jobs
- posthog - analytics

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
